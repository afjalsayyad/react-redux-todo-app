# Redux Toolkit Todo App

This is a simple Todo application built using React and Redux Toolkit.

## Features
- Add Todo
- Update Todo
- Delete Todo
- Global state management using Redux Toolkit

## Tech Stack
- React
- Redux Toolkit
- React Redux
- Vite
- Tailwind CSS

## Redux Flow
User action → Dispatch → Reducer → Store → UI update
